# LinkTree FreeToUse 🌐✨

Um agregador de links (*Linktree clone*) totalmente open-source, responsivo, moderno e leve, focado na privacidade e na customização total. Perfeito para colocar na bio do seu Instagram, TikTok, Twitter ou LinkedIn.

## 🚀 Demonstração & Recursos

- **Modo Escuro Nativo:** Design moderno e elegante feito para cansar menos os olhos.
- **100% Responsivo:** Adapta-se perfeitamente a computadores, tablets e smartphones.
- **Botão de Compartilhar:** Funcionalidade nativa em JavaScript que copia o link da página para a área de transferência do usuário com um clique.
- **Ícones Inclusos:** Integração com Font Awesome para exibição dos logos das principais redes sociais.
- **Fácil de Hospedar:** Pode ser publicado de graça em minutos usando o **GitHub Pages**, Vercel ou Netlify.

---

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído puramente com tecnologias web nativas, sem a necessidade de frameworks pesados:

*   **HTML5** – Estruturação semântica do conteúdo.
*   **CSS3** – Estilização customizada, efeitos de *hover* e layout responsivo.
*   **JavaScript (ES6)** – Lógica para o botão de compartilhamento de link.
*   **Font Awesome** – Biblioteca de ícones vetoriais.

---

## ⚙️ Como Customizar

Para deixar o projeto com a sua cara, basta clonar este repositório e alterar os seguintes arquivos:

1. **Sua Foto:** No arquivo `index.html`, mude a propriedade `src` da tag `<img>` para o link da sua foto (ou coloque uma foto na pasta e aponte o caminho).
2. **Seus Links:** No `index.html`, altere os atributos `href` das tags `<a>` com os links das suas redes sociais.
3. **Cores:** Se quiser mudar o visual, basta alterar as variáveis de cor no topo do arquivo `style.css`.
4. **Ícones:** Caso queira mudar os ÍCONES dos Links, basta localizar, por exemplo, "fa-github" e alterar para o de sua preferência, como: fa-google, fa-twitter, fa-instagram

---

## 📦 Como Hospedar no GitHub Pages (Grátis)

1. Vá até as **Settings** (Configurações) deste repositório no GitHub.
2. No menu lateral esquerdo, clique em **Pages**.
3. Em *Build and deployment*, selecione a branch `main` (ou `master`) e a pasta `/ (root)`.
4. Clique em **Save**. 
5. Pronto! Em poucos minutos, o GitHub gerará um link público para a sua bio.
